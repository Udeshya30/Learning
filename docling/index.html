Docling Feature Demo

This page describes the available `docling_ocr` features and shows how to test them from Python.

What this page covers

  • Docling feature set available from the installed `docling_ocr` package.
  • How to use the core class: SmolDoclingExtractor.
  • How to test extraction from images with Python.
  • Which configuration options are supported.

Core API

The main extractor in this package is SmolDoclingExtractor. It is a document understanding OCR model that works with scanned documents, forms, and layout-heavy images.

SmolDoclingExtractor
extract_text()
extract_text_from_image()
__call__()
device selection
model override

Features provided

  1. Image path extraction via extract_text(image_path, max_length=512).
  2. PIL image extraction via extract_text_from_image(image, max_length=512).
  3. Convenience call using extractor(image_input).
  4. Model selection with model_name while creating the extractor.
  5. Device control using device="cpu" or device="cuda:0".
  6. Generated text length can be limited with max_length.

Sample usage

Copy and run this Python sample to test text extraction from an image.

from docling_ocr import SmolDoclingExtractor

extractor = SmolDoclingExtractor(
    model_name="ds4sd/SmolDocling-256M-preview",
    device="cpu"
)

result = extractor.extract_text("path/to/sample-image.png", max_length=512)
print("--- Extracted text ---")
print(result)

Test instructions

1. Install required packages

pip install docling_ocr transformers pillow torch

2. Place a sample image

Use any document image file such as JPG, PNG, or scanned page. Put it in your workspace and update the path in the sample code above.

3. Run the sample script

python -c "from docling_ocr import SmolDoclingExtractor; extractor = SmolDoclingExtractor(device='cpu'); print(extractor.extract_text('your-image.png'))"

Advanced testing options

  • Change model: override model_name to use a different Hugging Face model.
  • Use GPU: set device="cuda:0" if CUDA is available.
  • Use PIL images: load the image in Python and call extract_text_from_image().
Note: This is a static demo page for learning. The actual extraction still runs in Python, not in the browser. Open this file in your browser to read the feature guide, then run the example code in your terminal.

Document layout to explore

If you want to test the package with a document that includes headings, tables, or form fields, use an image containing:

  • multiple text blocks
  • columns or table cells
  • captions or labels
